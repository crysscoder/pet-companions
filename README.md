# PetCompanions

Paper-плагин питомцев.

## Версии

- Java 17
- Gradle 8.8
- Paper API 1.20.1
- Lombok 1.18.34
- Authlib 3.13.56
- Plugin `1.0.0`

## Команды

- `/pet spawn`
- `/pet despawn`

## Сборка

```powershell
.\gradlew.bat clean build
```

Jar: `build/libs/PetCompanions-1.0.0.jar`

## Запуск тестового сервера

```powershell
.\gradlew.bat runServer
```
